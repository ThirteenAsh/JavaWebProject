## 项目说明
本项目为 Java Web 项目，基于 Spring Boot 开发，包含前后端分离的架构。

---

## 技术栈
- **前端**: Vue.js, Vite, JavaScript, CSS
- **后端**: Spring Boot, MyBatis, Java
- **数据库**: MySQL
- **构建工具**: Maven, npm
- **日志管理**: Logback

---

## 构建与运行

### 前端
1. 进入 `vue-frontend-management` 目录。
2. 安装依赖：`npm install`
3. 启动开发服务器：`npm run dev`
4. 构建生产环境代码：`npm run build`

### 后端
1. 进入 `web-management` 目录。
2. 使用 `mvnw` 构建项目：`./mvnw clean package`
3. 运行项目：`java -jar target/web-management-0.0.1-SNAPSHOT.jar`
